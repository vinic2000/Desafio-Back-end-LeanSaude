## Description

Aplicação foi feito usando nest.js como framework, prisma(ORM), redis(cache).

## Instalação

```bash
$ yarn install
```

## Executar aplicação

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# test coverage
$ yarn run test:cov
```
